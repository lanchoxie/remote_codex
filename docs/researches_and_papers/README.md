# Multi-Agent Research Papers

This directory stores the multi-agent and agent-swarm paper index collected in
this round. PDF files are treated as local research cache and are intentionally
ignored by Git to keep the repository small; run
`scripts/download_multi_agent_papers.ps1` to fetch them locally.

Naming rule: `YYYY-MM - Paper Title.pdf`

- The time prefix follows the first public appearance of the paper.
- The English title is kept, with characters removed only when they are invalid in Windows file names.
- Some files were downloaded from conference or journal PDFs, but the filename prefix still follows the first public release so the directory stays chronological.
